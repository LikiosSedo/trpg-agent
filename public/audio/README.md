# Audio Assets

Place MP3 files here. All should be loopable.

## BGM Tracks (required)
- `town-day.mp3` — 小镇白日（轻快、温暖）
- `town-night.mp3` — 小镇夜晚（安静、舒缓）
- `tavern.mp3` — 酒馆（温暖、木质感、轻快）
- `forest.mp3` — 森林探索（神秘、自然）
- `mines.mp3` — 矿道（压迫、低沉、回声）
- `wasteland.mp3` — 荒原（苍凉、空旷）
- `battle.mp3` — 普通战斗（紧张、节奏快）
- `boss-battle.mp3` — BOSS 战（史诗、激烈）
- `tension.mp3` — 悬疑紧张
- `sad.mp3` — 悲伤
- `triumph.mp3` — 凯旋/胜利
- `mystery.mp3` — 神秘
- `danger.mp3` — 危险逼近
- `peaceful.mp3` — 宁静温馨
