# Boss 战设计方案（v1 · 归档于 2026-04-17）

> **状态**：设计保留，**未开工**。优先级低于战斗系统基础体验修复。
> 等基础战斗 bug 修完后再回来看这份设计是否还合适。

## 背景

当前 boss 战（蛛母/暗影编织者/蚀日兽）在代码上与普通遭遇战走同一条 `startCombat()` 路径，只有 HP 阈值触发的硬编码 phase 2（召唤/分裂/狂暴），缺乏"战棋 boss 战"该有的**机制差异化**、**前置情报收集**、**演出层**。

## 设计核心：三层情报 × 三阶段战斗 × 演出层

### 一、三层情报（战前收集决定战斗难度）

复用现有 4 条解锁渠道（NPC 信任、Search、Chapter beat、Bestiary Hints），不造新系统，只在数据里补 boss 相关情报。

| 情报层级 | 作用 | 没收集的后果 |
|---|---|---|
| **L1 常识**（怕火/怕光/怕银） | 战斗自动 ×2 伤害 | 伤害普通,多打 2-3 回合 |
| **L2 机关知识**（卵囊/石柱/能量水晶） | 战斗中有 secondary objective 显示 | 被机关反复惩罚 |
| **L3 终极线索**（银器穿心 / 光辉刺入眼 / 冰封棱镜） | Phase 3 新形态可**一击秒杀** | 硬啃新形态 3-4 回合 |

玩家没收全也能打，但**收集越全，战斗越快、越帅**。情报是爽感倍率器。

### 二、三阶段战斗

**Phase 1（100% → 50%）· Arena Hazard**
- **蛛母 · 织暗者** = add 管理：场上 3 个卵囊（可攻击地形格）,不摧毁每 2 回合召唤小蜘蛛
- **暗影编织者** = 自愈打断：场上 4 根祭坛石柱,摧毁 3 根断 Shadow Mend 自愈
- **蚀日兽** = 位置控制：地图中心"虚空裂口",每 3 回合吸引所有单位向中心拉 1 格

**Phase 2（50% → 20%）· 读条 + 强化**
- HP 降到 50% 不立即变身,改成 **2 回合读条**
- 前端 charging bar: `"蛛母：毒雾领域 ■■□□"`
- 玩家 3 选 1：打断(集中输出) / 走位(拉开减伤) / 道具(情报解锁的"月光石"定身跳过 1 回合)
- 读条完成 → 进入 Phase 2 强化(现有召唤/分裂/狂暴演出化)

**Phase 3（20% → 0）· 真·新形态（残血演出）**
- HP 清零瞬间 **"临死挣扎"不死**,从地上立起新形态,血条刷新至 40%
- 同时 **属性剧变**：蛛母 8 眼全开、暗影实体化穿墙、蚀日兽崩解为能量形态
- 如果玩家装备了 L3 情报解锁的**终极武器**(银器 tag / 光辉 tag)命中 → **一击秒杀**（戏剧处决）
- 没有则 Phase 3 = 再打 40% 血条 + 全新技能套

### 三、演出层

**新增 TurnEvent**: `boss_phase_transition { bossName, fromPhase, toPhase }`
- 代码触发 HP 阈值
- DM 收到意图写 20-40 字特写镜头("蛛母八条腿齐齐抬起,卵囊蠕动发出密集脆响,血红的眼睛逐一睁开...")
- 前端：画面闪红 + 镜头推近 1.5s + 特写文字居中 → 回战斗

**Phase 3"临死挣扎"特殊演出**：
- 血条归零瞬间一段长镜头("你以为它倒下了 —— 它的核心裂开一道血色光芒,慢慢又立了起来...")
- 视觉上：画面极速震动 + 血色粒子 + 镜头拉远再推近,玩家印象极深

### 四、情报如何"兑现"到战斗中

- L3 情报解锁 = 玩家拿到某个 flag(`intel_silver_kill_spider` 等)
- 同时开启 **前置准备的物品/装备**：银匠铺才出现"订制银剑"选项、商人才出现"月光石"、任务奖励"圣水"
- 战斗中：命中 + Phase 3 + weapon 带对应 tag → 触发秒杀逻辑
- 战斗 UI 角落显示 **"已解锁情报"气泡**：战斗中能看到自己收集了啥

## 时长估算

| 玩家状态 | P1 | P2(读条+强化) | P3 | 总回合 | 时长 |
|---|---|---|---|---|---|
| 情报全 | 3-4 | 2+1-2 | 1 | 7-9 | 5-7 min |
| 情报半 | 4-5 | 2+3 | 2-3 | 11-13 | 9-11 min |
| 情报零 | 5-6 | 2+3 | 3-4 | 13-15 | 11-13 min |

## 工程分波

**Wave 1（MVP · 蛛母一个 boss 端到端）**
- `combat-grid` 加"可攻击物件"（卵囊格）+ 每回合 hook
- 通用 `MonsterPhase` 数据结构取代硬编码 phase2 flag
- 蛛母三阶段 + 3 条情报 + 银剑物品
- 验证"情报 → 战斗兑现"设计是否成立

**Wave 2（演出 + 读条）**
- `boss_phase_transition` 事件 + DM 特写 prompt 片段
- Pending action 读条 state + 前端 bar + 可打断机制
- 蛛母演出完整化

**Wave 3（三 boss 横向复制）**
- 暗影编织者：自愈打断 + 光辉一击
- 蚀日兽：位置控制 + 冰封棱镜
- 战斗 UI 角落"已解锁情报"提示气泡
