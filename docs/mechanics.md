# 游戏机制设计

> 基于 D&D 5e SRD 简化，适配 CLI 文字冒险场景。

## 1. 角色属性

采用 D&D 经典六维属性，数值范围 3–20（普通人 10，冒险者通常 8–16）。

| 属性 | 缩写 | 用途 |
|------|------|------|
| 力量 Strength | STR | 近战攻击、负重、破门 |
| 敏捷 Dexterity | DEX | 远程攻击、闪避、潜行、先攻 |
| 体质 Constitution | CON | 生命值、体力、毒抗 |
| 智力 Intelligence | INT | 奥术、调查、记忆 |
| 感知 Wisdom | WIS | 察觉、洞察、追踪、医疗 |
| 魅力 Charisma | CHA | 说服、欺骗、威吓、表演 |

**属性修正值** = `Math.floor((属性值 - 10) / 2)`

## 2. 技能检定

### 核心机制：d20 + 修正值 vs DC

```
检定结果 = d20 + 属性修正值 + 熟练加值(如有)
```

- **DC (难度等级)**：DM 根据情境设定
  - 5 = 很简单 | 10 = 简单 | 15 = 中等 | 20 = 困难 | 25 = 极难
- **熟练加值**：固定 +2（简化，不随等级成长）
- **自然 20**：自动成功，可能触发特殊效果
- **自然 1**：自动失败，可能触发负面效果

### 常用技能列表

| 技能 | 关联属性 | 场景示例 |
|------|----------|----------|
| 运动 Athletics | STR | 攀爬、游泳、推搡 |
| 体操 Acrobatics | DEX | 平衡、翻滚、闪躲陷阱 |
| 巧手 Sleight of Hand | DEX | 偷窃、开锁、藏物 |
| 隐匿 Stealth | DEX | 潜行、埋伏 |
| 调查 Investigation | INT | 搜索线索、解谜 |
| 奥秘 Arcana | INT | 辨识魔法、解读符文 |
| 察觉 Perception | WIS | 发现隐藏事物、听动静 |
| 洞察 Insight | WIS | 判断对方是否说谎 |
| 医疗 Medicine | WIS | 急救、判断伤势 |
| 说服 Persuasion | CHA | 交涉、请求 |
| 欺骗 Deception | CHA | 撒谎、伪装 |
| 威吓 Intimidation | CHA | 恐吓、逼问 |

### 对抗检定

当两方互相对抗时（如 NPC 洞察 vs 玩家欺骗），双方各掷一次，高者胜。

```
玩家欺骗: d20 + CHA修正 + 熟练(如有)
NPC洞察:  d20 + WIS修正 + 熟练(如有)
```

## 3. 战斗系统

### 3.1 战斗流程

```
开始战斗
  ↓
先攻掷骰 (d20 + DEX修正) → 排出行动顺序
  ↓
┌──────────── 回合循环 ────────────┐
│  当前角色的回合:                   │
│    1. 移动 (最多 30ft / 6格)      │
│    2. 动作 (攻击/施法/使用物品等)   │
│    3. 附赠动作 (如有)              │
│  ↓                                │
│  下一角色...                      │
│  ↓                                │
│  所有角色行动完毕 → 新一轮          │
└───────────────────────────────────┘
  ↓
战斗结束条件: 一方全灭 / 逃跑 / 投降 / 剧情触发
```

### 3.2 攻击判定

```
攻击掷骰 = d20 + 属性修正(STR或DEX) + 熟练加值
```

- 结果 ≥ 目标 AC (护甲等级) → 命中
- **自然 20** → 暴击，伤害骰翻倍
- **自然 1** → 必定未命中

### 3.3 伤害计算

```
伤害 = 武器伤害骰 + 属性修正值
暴击伤害 = 武器伤害骰 × 2 + 属性修正值
```

常用武器伤害骰:

| 武器 | 伤害 | 属性 | 类型 |
|------|------|------|------|
| 匕首 | 1d4 | DEX | 穿刺 |
| 短剑 | 1d6 | DEX | 穿刺 |
| 长剑 | 1d8 | STR | 挥砍 |
| 大剑 | 2d6 | STR | 挥砍 |
| 短弓 | 1d6 | DEX | 穿刺 |
| 长弓 | 1d8 | DEX | 穿刺 |
| 木棒 | 1d4 | STR | 钝击 |
| 战锤 | 1d8 | STR | 钝击 |

### 3.4 护甲等级 (AC)

```
无甲: AC = 10 + DEX修正
轻甲: AC = 甲值 + DEX修正
中甲: AC = 甲值 + DEX修正(上限+2)
重甲: AC = 甲值 (无DEX加值)
+ 盾牌: AC +2
```

### 3.5 生命值 (HP)

```
初始HP = 体质骰 + CON修正
```

- HP ≤ 0 → 倒地、濒死
- 简化规则：HP 降到 0 时直接失去意识，队友可用医疗检定(DC 10)稳定

## 4. 简化法术系统

### 法术位

简化为 3 个等级的法术位：

| 法术等级 | 法术位数 | 示例 |
|---------|---------|------|
| 戏法 (0环) | 无限 | 法师之手、光亮、冷冻射线 |
| 1 环 | 3 | 魔法飞弹、治疗术、雷鸣波 |
| 2 环 | 2 | 隐形术、蛛网术、次级复原术 |

法术位在长休息后恢复。

### 施法检定

部分法术需要法术攻击掷骰或目标进行豁免检定：

```
法术攻击 = d20 + 施法属性修正 + 熟练加值
法术DC = 8 + 施法属性修正 + 熟练加值
```

## 5. 休息与恢复

| 类型 | 时间 | 效果 |
|------|------|------|
| 短休息 | 叙事中的片刻 | 恢复 HP (掷生命骰) |
| 长休息 | 叙事中的安全休整 | 恢复全部 HP + 全部法术位 |

## 6. 经验与成长

简化处理——不追踪 XP，改用 **里程碑升级**：

- 完成关键任务节点后，DM Agent 判断是否升级
- 升级效果：HP 增加、熟练加值不变(保持简单)、可能解锁新能力

## 7. 物品与经济

### 货币

统一使用 **金币 (gp)** 作为唯一货币单位。

### 物品分类

| 类型 | 示例 | 效果 |
|------|------|------|
| 消耗品 | 治疗药水、解毒药、火把 | 使用后消失 |
| 装备 | 武器、护甲、盾牌 | 穿戴后提供持续效果 |
| 关键物品 | 任务道具、信件、钥匙 | 推动剧情，不可丢弃 |
| 杂物 | 绳索、铁钉、粉笔 | 特定场景下有创意用途 |

### 负重

简化为 **背包格子系统**：玩家有 10 格背包空间，每件物品占 1 格（大型物品占 2 格）。

## 8. DM 裁决原则

这是一个由 AI Agent 驱动的 TRPG，DM Agent 在执行规则时应遵循：

1. **规则服务于叙事**：如果规则和精彩故事冲突，选择故事
2. **鼓励创意**：玩家提出规则外的创意行动时，设定合理 DC 让他们尝试
3. **失败也有趣**：检定失败不是"什么都没发生"，而是"发生了意想不到的事"
4. **保持节奏**：战斗不超过 5 轮，非战斗检定不连续超过 3 次
5. **难度自适应**：根据玩家表现动态调整 DC 和怪物行为
