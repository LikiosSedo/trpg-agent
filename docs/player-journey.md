# 玩家旅程设计（Player Journey）

## 设计原则

1. **渐进揭示** — 世界不是一次性展开的，而是通过交互一层层剥开
2. **NPC 引导** — 关键信息由 NPC 主动告知，不靠玩家猜
3. **选择有意义** — 每个决策影响后续剧情走向
4. **代入感优先** — 先让玩家"成为角色"，再开始冒险

---

## 第一幕：醒来（开场 5 分钟）

### 场景
玩家在一辆颠簸的马车上醒来，浑身湿透，头痛欲裂。身上只有一把生锈的短剑和一个空钱袋。记忆模糊——只记得自己叫什么名字，以及"有人让我去破晓镇"。

### DM 引导
```
DM: 你在一辆颠簸的货运马车上醒来。雨水从破旧的油布缝隙渗进来，
冷得你牙齿打颤。赶车的老头回头看了你一眼：

老头: "醒了？破晓镇到了。你浑身是伤倒在路边，是我把你捡上来的。"

（马车在一座石头牌坊前停下，牌坊上刻着"破晓镇"三个模糊的字。
远处能看到几点灯火和一座石碑的轮廓。雨很大。）

DM: 你叫什么名字？你还记得自己是做什么的吗？
```

### 玩家选择：角色创建
- 输入名字
- 选择职业（剑士/法师/游侠/牧师 — 影响初始属性和技能）
- 简述记忆碎片（"我记得一把剑和火焰" / "我记得有人叫我的名字"）

### 系统行为
- 根据职业分配属性和初始技能
- 记忆碎片成为后续伏笔（DM 会在适当时候回应）
- 失忆设定 = 玩家不需要知道整个世界观，和角色一起探索

---

## 第二幕：酒馆之夜（10-20 分钟）

### 自然引导链

```
下雨 + 受伤 → 需要找个地方避雨
唯一亮着灯的地方 → 碎盾亭酒馆
进门 → 格雷格主动搭话
```

### 格雷格的引导对话

格雷格不会等玩家问——他会主动推进：

1. **阶段1：关心**（信任0）
   - "浑身湿透的，坐下，先喝口热汤。" （免费给一碗汤 — 建立好感）
   - "你是外地来的？这大晚上的谁会来破晓镇……"

2. **阶段2：情况介绍**（自然聊天中）
   - "镇上最近不太平。矿洞里出了怪事，矿工都不敢下去了。"
   - "要是你能打架的话，明天去冒险者公会看看。他们在招人。"
   - → **玩家获得方向：去公会找艾琳娜**

3. **阶段3：小莉出场**（格雷格说话时）
   - 小莉端着盘子路过，突然停住看玩家一眼，说："你身上有……奇怪的味道。"
   - 格雷格："别胡说，去干活。"
   - → **伏笔：小莉的感知能力**

4. **阶段4：住宿**
   - "楼上有间空房，今晚免费。明天你看着给就行。"
   - → **玩家有了据点（酒馆楼上的房间）**

### 触发条件
- 玩家说任何关于"受伤"的话 → 格雷格叫小莉去拿绷带
- 玩家问"矿洞" → 格雷格表情微变，转移话题
- 玩家问"石碑" → 格雷格："那东西从我爷爷那辈就在了，没人看得懂。"

## 第三幕：第一天（20-40 分钟）

### 早晨探索引导

DM 描述清晨的破晓镇：
```
DM: 雨停了。清晨的阳光照在广场中央的石碑上，
符文似乎在发光。镇上的人开始出来干活。

你从酒馆楼上的窗户可以看到：
- 北边：通往矿洞的路，几个矿工在犹豫要不要进去
- 东边：集市渐渐热闹起来，有个矮人铁匠在开门
- 南边：远处的森林笼罩在薄雾中
- 广场上：一个银发女性站在石碑前，似乎在研究什么
```

### 引导到公会（艾琳娜）

不管玩家走哪条路，都会被引导到公会：
- **去矿洞** → 矿工拦住："你也是公会派来的？去找会长吧。"
- **去集市** → 铁匠格罗姆："你要装备？公会新人有折扣。"
- **去森林** → 路上遇到受伤的猎人："别去，太危险了。去公会报告！"
- **看石碑** → 遇到艾琳娜（直接触发对话）

### 艾琳娜的引导

1. **任务介绍**：
   - "矿洞里两周前失踪了两个矿工。搜救队也没回来。"
   - "公会需要人进去看看。但我必须先确认你的能力。"

2. **第一个任务：森林试炼**（安全的新手教学关卡）
   - "暮色森林边缘最近有狼群骚扰伐木工。去处理一下，顺便带回猎人老林的近况报告。"
   - → **教玩家：移动、战斗、搜索、与NPC对话**

3. **装备支持**：
   - "去铁砧铺找格罗姆，报我的名字，新人第一件装备半价。"
   - → **引导玩家到商店系统**

## 第四幕：森林试炼（30-60 分钟）

### 教学目标
每个遭遇教会玩家一个系统：

| 遭遇 | 教什么 | 怎么教 |
|------|--------|--------|
| 森林路上的岔路 | **探索 + Look** | 两条路选一条，DM 描述不同 |
| 3只狼 | **战斗系统** | 简单战斗，DM 教先攻、攻击、伤害 |
| 旧伐木场门口 | **Search** | 搜索发现哥布林陷阱 |
| 哥布林营地 | **选择** | 战斗 or 潜行 or 谈判？ |
| 猎人老林 | **NPC 对话** | 他告诉你动物在逃离矿洞方向 |
| 月池 | **伏笔** | 水面映出一个不属于你的影子 |

### 回镇报告

回到公会后，艾琳娜根据你的表现给出评价：
- 全杀 → "你很能打。但冒险者不只是杀手。"
- 潜行/谈判 → "好判断。不是所有问题都需要剑来解决。"
- 找到月池线索 → "……有意思。" （好感大幅提升）

**获得**：
- 经验值 + 等级提升到 2 级
- 公会正式会员资格
- 下一个任务：调查矿洞上层

## 渐进揭示时间表

| 时间 | 玩家知道的 | 玩家不知道的 |
|------|-----------|-------------|
| 第一幕 | 自己失忆了，到了一个小镇 | 一切 |
| 第二幕 | 镇上不太平，矿洞有问题 | 暗影教团、虚空棱镜、NPC秘密 |
| 第三幕 | 有个公会，有任务可以做 | 教团渗透了镇子 |
| 第四幕 | 森林里有异常，动物在逃离 | 月池的仪式痕迹含义 |
| 后续 | 矿工失踪和暗影有关 | 维克多被胁迫，卡恩的真实身份 |
| 中期 | 暗影教团的存在 | 虚空棱镜的具体能力 |
| 后期 | 完整真相 | 最终决战的选择 |

---

## DM 引导规则

### 必须做的
1. **新玩家的前3轮，DM 主动推进剧情**（不等玩家找方向）
2. **每个 NPC 在首次见面时给出一个明确的下一步指引**
3. **每次探索结束回到酒馆时，格雷格或小莉提供一句情报**

### 不能做的
1. **不能一次性倒出所有信息**（"这个镇子有暗影教团..." ❌）
2. **不能让玩家迷路超过2轮**（如果玩家不知道该干嘛，NPC 主动出现引导）
3. **不能跳过战斗教学**（第一次战斗必须详细解释规则）

### 防卡机制
如果玩家连续2轮没有明确行动：
- 酒馆场景 → 小莉跑来说"有人在找你！"
- 镇上场景 → 一个NPC路过搭话
- 森林场景 → 发现脚印或听到声音
- 矿洞场景 → 矿壁上的符文发出微光
