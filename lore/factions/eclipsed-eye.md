# 蚀目者教团

一个崇拜/服务被遗忘的暗影位面实体"蚀目者"的秘密组织。破晓镇当前危机的幕后推手。

## 外部认知

在第二章之前，"蚀目者"这个名字对大多数镇民来说只是一个古老的恐怖故事——"如果你不听话，蚀目者就会来找你"。没有人真的相信它存在。

冒险者公会的档案里有一条简短的记录："蚀目者信徒——约三百年前活跃于灰脊山脉地区的暗影教团，已被联合讨伐军剿灭。"之后再无更新。

## 组织架构

| 层级 | 代号 | 角色 |
|---|---|---|
| 首领 | 蚀语者 | 掌握完整激活仪式，远程指挥，极少露面 |
| 执行者 | 蚀弦（当前：卡恩） | 外勤精英，负责关键任务执行和心理操控 |
| 外围成员 | 无代号 | 渗透到镇内各处的棋子（叶绿的助手等） |
| 受控者 | 无代号 | 被胁迫服务但非自愿的对象（维克多） |

## 当前目标

**核心目标**：挖掘并激活灰脊矿道深层的**虚空棱镜**，撕裂位面壁垒，召唤蚀目者的虚空先驱。

**执行步骤**：
1. ✅ 控制镇长 → 获取矿道深层通行许可
2. ✅ 爆破中层封堵 → 打通通往深层的通道（三周前完成，造成"矿难"掩护）
3. 🔄 找到"灵视者"（导管） → 卡恩正在缩小范围
4. ⏳ 在下一次日食窗口执行激活仪式

## 他们的标记

一个被一道斜线划过的眼睛——代表"被蒙蔽的眼睛终将睁开"。这个符号出现在：
- 叶绿助手抽屉里的纸条上
- 维克多收到的威胁信上
- 矿道中层秘密集会室的墙壁上
- 碎石荒原废弃瞭望塔的遇难冒险者笔记里

## 教团与沈织

十五年前，教团抓获了白塔学院的位面理论学者沈织——当时唯一能读懂棱镜铭文的活人。沈织被迫翻译了大部分铭文，但在逃跑前暗中篡改了关键段落，导致教团至今掌握的仪式流程**不完整**。

这就是为什么教团花了十多年仍在"准备"——他们不知道自己的仪式文本被动过手脚。卡恩接手后重新翻译了一部分（他找到了变体版本），但标准版仍有缺口。

## 教团为什么选择破晓镇

不是随机的。虚空棱镜就在灰脊矿道深层——三千年前被银曜精灵封印。晨光石碑是封印锚点。教团需要：
1. 破坏封印（让维克多删除封印记录 + 等待日食中断光柱）
2. 物理接近棱镜（爆破矿道 + 清除阻碍）
3. 找到灵视者作为仪式导管

三个条件都只能在破晓镇完成。
